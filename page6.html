1969 - ARPAnet

  • In 1958, the Advanced Research Projects Agency (ARPA) was launched and is known today as the Defense Advanced Research Projects Agency (DARPA). ARPAnet was the first network to run on packet-switching. On the 29th October 1969, computers at Stanford and UCLA connected for the first time. 1969 - Unix

  • During the 60's, the Unix operating system was established. This operating system has heavily influenced the current Linux and FreeBSD operating systems.

1971 - Email

  • In 1971, Ray Tomlinson created the Email. He also implemented the "@" symbol which separates the user name from the computer name.

1971 - Project Gutenberg

  • In 1971, Project Gutenberg was developed. Project Gutenberg provides books and documents on their website that can be accessed by anyone for free.

1974 - TCP/IP

  • In 1974, A proposal was published to link Arpa networks together into an “inter-network”, which would have no central control and would work around a transmission control protocol (which eventually became TCP/IP).

1979 - Multiplayer games

  • In 1979, a predecessor to the the games World of Warcraft and Second Life was created. The game was called MUD (MultiUser Dungeon). These MUDs were completely text-based virtual worlds which had multiple elements: role-playing games, interactive, fiction, and online chat.

1982 - The first Emoticon ever

  • The first emoji that was created was the smiley face. It was created in 1979 by Kevin MacKenzie. In 1982, Scott Fahlman used this emoticon after a joke.

1983 - ARPAnet switches to TCP/IP

  • The ARPAnet computers had to switch over to the TCP/IP protocols made by Vinton Cerf before the 1st of January 1983. The TCP/IP protocols were also developed in 1983.

1984 - Domain Name System

  • In 1984, the domain name system as well as the first domain name servers were created. This system was made so that the addresses on the internet were more human friendly and thus easier to remember compared to the numerical ip address. The DNS server converts the domain name into the ip address of the website.

1986 - Protocol wars

  • The Protocol wars began during the year 1986. European countries developed their own protocol - Open Systems Interconnection (OSI) as an alternative to the TCP/IP protocol used by the United States. Eventually thee TCP/IP protocol won.

1988 - Internet Relay Chat

  • Internet Relay Chat was first deployed in 1988, which served as a precursor for the real-time chat and messaging programs we use today.

1988 - First major malware attack on the internet

  • In 1988, Robert Tappan Morris released the first major Internet worm which interrupted large parts of the internet. This worm is referred to as "The Morris Worm"

1989 - The idea of the World Wide Web

  • In 1989, a proposal for the creation of the World Wide Web was published. The World Wide Web was initially called Mesh, but later on the term "World Wide Web" was made up while Tim Berners-Lee was still developing the code for it.

1991 - MP3

  • In 1991, the MP3 file format became the standard and a popular file type for sharing audio and songs.

1993 - Mosaic Browser

  • Although it wasn't the first web browser, Mosaic was the most downloaded internet browser back then. Following Mosaic, Netscape Navigator was released a year later. This marked the start of the browser wars that would ensue for the next decade.

1998 - Google

  • In 1998, the Google search engine was released and it dramatically altered the way people can find information online.
